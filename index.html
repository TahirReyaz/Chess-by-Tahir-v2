RULES AND INSTRUCTIONS

If the game doesn't load properly: press an arrow button from the CONTROL BOX given below

Use the control buttons provided below the Chess board or arrow keys to move the cursor

Click on the Enter button or Enter key form your keyboard to select a chess piece and move the cursor to desired location

Again press the Enter button or Enter key from your keyboard to place the selected piece

The is no restriction in choosing a peice and its location

If there is a peice already at the location where you put the new selected piece,

then the old piece will move to Trash
